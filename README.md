# java-web-sbt-blade-api

## Description
A POC for blade framework.

## Tech stack
- cors
- middleware
- endpoints
    - get, put, delete

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
